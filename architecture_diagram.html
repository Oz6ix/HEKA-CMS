    participant M as Middleware
    participant C as AppointmentController
    participant S as AppointmentService
    participant R as AppointmentRepository
    participant DB as SQLite

    B->>M: GET /admin/appointment
    M->>M: Auth + EnsurePermissions
    M->>C: index()
    C->>S: getAppointments()
    S->>R: getAll()
    R->>DB: SELECT * FROM appointments
    DB-->>R: Results
    R-->>S: Collection
    S-->>C: Formatted Data
    C-->>B: Render modern layout + data

    Note over B,DB: Print Flow
    B->>C: GET /appointment/print/{id}
    C->>S: getById(id)
    S->>R: find(id)
    R->>DB: SELECT with relations
    DB-->>C: Appointment + Prescriptions + Tests
    C-->>B: Professional HTML (letterhead + Rx table)
